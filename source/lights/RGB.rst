RGB Light
===============================

Shared Positve with three negative


* Voltage: DC 24V
* Power: 50W
* LED: 280 RGB 
* Aviation connector: G16-4


.. image:: ../image/rgbpanel.png

.. admonition:: 

    It works with Constent Current controler. 
    Make sure the current is not above 0.8A for each channel.
